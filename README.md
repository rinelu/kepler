# Kepler

A 3D planet simulator (C + raylib)

This is **work in progress.**

## Building

Build:

```sh
make
```

Run:

```sh
make run
```

Clean:

```sh
make clean
```

## Versioning

Kepler follows **Semantic Versioning**:

```
MAJOR.MINOR.PATCH
```

See the version history for detailed release notes.
